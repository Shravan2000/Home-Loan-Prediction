HOME LOAN PREDICTION SYSTEM

Aim:
   To help the bank customers to know whether their home loan can be approved or not.

Algorithm used:
Random Forest

Algorithms tried:
K nearest neighbors
SVM
Logistic Regression

How To Run:
Run the project by using the command,
python app.py

Requirements:
python3
scikit-learn
flask
pandas
numpy
pickle

Result:
   Out of all algorithms, random forest algorithm gave the best result(an accuracy of 81%).
